---
title: CleanupService
description: Service for repository cleanup operations
---

The CleanupService provides methods for cleaning up repositories after operations like BFG Repo-Cleaner.

## ApplyBfgObjectMapStream

Applies a BFG object map to update references after repository rewriting.

### Request (Stream)

<ParamField path="repository" type="Repository" required>
  The repository to apply the object map to. Only available in the first message.
</ParamField>

<ParamField path="object_map" type="bytes">
  Raw object-map file data as generated by [BFG Repo-Cleaner](https://rtyley.github.io/bfg-repo-cleaner/). Each line contains two space-separated object SHAs: the original SHA and the SHA after BFG rewrote the object.
</ParamField>

### Response (Stream)

<ResponseField name="entries" type="Entry[]">
  Array of parsed entries from the object map
</ResponseField>

#### Entry

<ResponseField name="type" type="ObjectType">
  The type of Git object (commit, tree, blob, tag)
</ResponseField>

<ResponseField name="old_oid" type="string">
  The original object ID before rewriting
</ResponseField>

<ResponseField name="new_oid" type="string">
  The new object ID after rewriting
</ResponseField>
